Java Container(List)

why we use container

Sometimes, we only know exactly number and type of the objects when we run the code. So we need container to allow us to create any number of objects at any time.

List

List contains ArrayList and LinkedList

ArrayList

ArrrayList base on dynamic array, arrays are stored sequentially when storing data, and memory for storing data is also contiguous. When we read a item, we just need tell where it is (the index) and we will get it (it will cost constant time). And if we want insert or delete will cost a lot of time.

For insert, if we want to insert a item into the array. We need change all the index after the index that we insert(index adds 1).

For delete, we can find the item we want to delete very quickly, but when we delete it, we need to change all the index after this index(index minus 1)

But if we just add or delete an item at the end of the array, it only takes a constant time, since it doesn’t need change any index.

LinkedList

LinkedList is a two-way list, that means any item in the list we record two address, one store the address of the previous item, the other one store the address of the next item. With this method, we add item into the array will cost constant time, for we only need to change two items’ address.

But for searching, if we want to get a item, we need to search the all list for the random access. For every find, we need start with the first item and according the address, we can find the next one and the next one….. it will cost O(n) time.

Functions

contains()

we can use this function to determine if an object is in the list.

remove()

If we want to delete an object, we can give the object reference to this method. Or give it the object index will server same function.

IndexOf()

We can use it to get an object index.

subList(index1,index2)

It allows us to create a fragment of our list, from index1 to index2.

And be careful, subList returns only a view, and any changes to the returned list are reflected in the original list, and vice versa.

containsAll()

Give a list to this function, and as long as you have an object is not in the original list, return false.

removeAll()

Give a list, and remove all object in this list in the original list.

set(index,object)

Replaces the object with the given object at the specified index.

isEmpty()

Return true if the list is empty.

clear()

Clear the whole list.

toArrary()

return a array that contains all the object in the list.

Like Object [] o = list.toArray()
